AMFI NAV API

Retrieve NAV data for mutual funds with ease!

Introduction

Welcome to the AMFI NAV API! This API provides a convenient way to retrieve Net Asset Value (NAV) data for various mutual funds. Use the provided endpoint to get the latest NAV or specify parameters for more specific queries.

How to Use

To retrieve NAV data for mutual funds, use the following endpoint: /nav_data/

You can provide the following parameters:

  • scheme_code (mandatory, regex: ^\d{6}$)
  • isin (optional, regex: ^[A-Za-z0-9]{12}$)
  • scheme_name (optional)
  • date (optional)
  • page (optional, default: 1)
  • page_size (optional, default: 10, max: 100)

Example usage: /nav_data/?scheme_code=119551&date=2024-01-15

Additional Details

This API supports various parameters to cater to your specific needs. Whether you are looking for NAV data for a particular scheme or want historical data for a specific date, this API has you covered.

Feel free to explore the options and tailor your requests accordingly. For any issues or questions, please refer to the documentation or contact our support team.
